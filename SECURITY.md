# Security Policy

## Responsible Use

HackMate is an educational tool for personal use. Users are solely responsible for ensuring their use complies with applicable laws and third-party terms of service.

## Reporting a Vulnerability

If you discover a security vulnerability in HackMate, please open a GitHub issue labeled `security`. Do not include sensitive details publicly — describe the general nature of the issue and we will follow up.
