HackMate
Automates the entire OpenCore hackintosh USB process. No manual config.plist editing.
Download EXE (Windows) ⭐ Star on GitHub
What it does
  • → Scans CPU, GPU, audio, ethernet, WiFi, NVMe, Thunderbolt
  • → Shows compatible macOS versions for your hardware
  • → Formats USB, downloads macOS recovery directly from Apple
  • → Generates SMBIOS, config.plist, SSDTs, kexts automatically
  • → Built-in config.plist editor with iGPU framebuffer + audio layout suggestions
  • → Uses the exact same tools as the Dortania guide — no special tricks

FAQ
My antivirus flagged HackMate.exe
False positive — PyInstaller-bundled executables trigger heuristic scanners. Every major AV (Defender, Kaspersky, ESET, CrowdStrike) reports clean. The EXE is built transparently on GitHub Actions from this source code.
Does it work on Alpine / musl libc?
Yes — it's pure Python with no native extensions that would break on musl.
Is it safe to run?
It only touches your target USB drive. Internal disks are hidden from the device list. Source is fully open on GitHub.
Do I still need to follow the Dortania guide?
HackMate automates the Dortania guide steps using the same tools (macrecovery, SSDTTime, OpenCore). Reading the guide is still useful for understanding what's happening, but you don't need to run any of it manually.
